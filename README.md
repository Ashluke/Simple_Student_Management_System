# Student Management System

A full-stack Student Management System built with **FastAPI (Python)** for the backend and **TypeScript (Vanilla)** for the frontend.  
It allows managing classes, students, and different types of assessments (scores, attendance, exams, and grades).

---

# Features

- Create and manage classes
- Add and remove students per class
- Record assessments:
  - Test Scores
  - Attendance tracking
  - Exam scores (midterm & endterm)
  - Grades (midterm, endterm, final, GWA)
- Configurable max scores and attendance dates per class
- Persistent data using database + JSON fields

---

# Tech Stack

## Backend
- FastAPI
- SQLAlchemy
- Pydantic

## Frontend
- HTML
- CSS
- TypeScript (compiled via `tsc`)
- Vanilla JavaScript (output)

# Installation

## 1. Backend setup (FastAPI)

```bash
cd backend

python -m venv venv

venv\Scripts\activate   # Windows

pip install -r requirements.txt
```

---

## 2. Frontend setup (TypeScript)

```bash
cd frontend

npm install

npm run build
```

---

## 3. Run backend server
```bash
uvicron main:app --reload
```

# Usage

1. Finish installation
2. Manage classes and students
3. Input assessments and configs dynamically

---

# Key Idea

This system separates:

- Assessment data (per student)
- Configuration data (per class)

So max scores, attendance dates, etc. are stored once per class тАФ not repeated per student.
